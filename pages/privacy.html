Политика конфиденциальности

Последнее обновление: 1 января 2025 г. | ID документа: JRZ-PRIVACY-001

1. Введение

Jorenza («мы», «нас», «наш») уважает вашу конфиденциальность. Эта Политика описывает, какие данные мы собираем, как их используем и как защищаем.

Jorenza спроектирован с принципом «минимум данных» — мы собираем только то, что необходимо для работы Сервиса.

2. Данные, которые мы НЕ собираем

В отличие от большинства сервисов, Jorenza не собирает:

  • Имена и фамилии пользователей
  • Адреса электронной почты (без регистрации)
  • Пароли или токены GitHub
  • Содержимое файлов репозиториев
  • IP-адреса в постоянных логах
  • Cookies для трекинга

3. Данные, которые мы обрабатываем

3.1 URL репозиториев

Когда вы вставляете ссылку на GitHub репозиторий, мы обращаемся к публичному GitHub API для получения метаданных (название, описание, статистика). Этот запрос не требует авторизации и не передаёт нам личные данные.

3.2 Локальное хранилище браузера

История деплоев сохраняется в localStorage вашего браузера. Эти данные хранятся только на вашем устройстве и никогда не передаются на наши серверы.

3.3 Технические логи

Стандартные веб-сервер логи могут содержать: время запроса, тип браузера, запрошенный URL. Логи хранятся не более 7 дней и используются исключительно для диагностики технических проблем.

4. Третьи стороны

GitHub API

Jorenza использует публичный GitHub REST API. При каждом запросе GitHub может фиксировать ваш IP-адрес согласно собственной политике конфиденциальности (docs.github.com/privacy).

Google Fonts

Интерфейс загружает шрифты через Google Fonts CDN. Google может логировать запросы шрифтов. Вы можете заблокировать это через соответствующие браузерные расширения.

5. Безопасность данных

Поскольку Jorenza не хранит персональные данные на серверах, риск утечки данных минимален. Все соединения используют HTTPS-шифрование.

История деплоев в localStorage защищена стандартными механизмами браузерной безопасности (same-origin policy).

6. Ваши права

Вы имеете право в любой момент:

  • Очистить историю деплоев (через настройки браузера → очистить localStorage)
  • Запросить информацию о любых данных, связанных с вами
  • Потребовать удаления любых данных

Для запросов пишите: privacy@jorenza.app

7. Изменения политики

При существенных изменениях политики конфиденциальности мы уведомим пользователей на главной странице. Дата последнего обновления указана в заголовке этого документа.
