Условия использования

Последнее обновление: 1 января 2025 г. | ID документа: JRZ-TERMS-001

1. Принятие условий

Используя платформу Jorenza («Сервис»), вы соглашаетесь с настоящими Условиями использования. Если вы не согласны с какими-либо положениями, пожалуйста, прекратите использование Сервиса.

Jorenza — это инструмент для анализа публичных GitHub репозиториев и предоставления удобных ссылок на их файлы. Сервис не хранит файлы проектов и не является хостинг-провайдером.

2. Описание сервиса

Jorenza предоставляет следующие услуги:

  • Анализ публичных GitHub репозиториев через официальный GitHub API
  • Генерацию уникальных URL-идентификаторов для проектов
  • Создание коротких ссылок для удобного шеринга
  • Отображение структуры файлов и прямых raw-ссылок на файлы
  • Лендинг-страницы проектов с метаданными репозитория

3. Допустимое использование

Пользователь обязуется:

  • Использовать Сервис только для публичных репозиториев, на которые имеет право ссылаться
  • Не нарушать авторские права третьих лиц
  • Не использовать Сервис для автоматизированных запросов, превышающих разумные лимиты
  • Не пытаться обойти технические ограничения Сервиса
  • Соблюдать Условия использования GitHub (github.com/site/terms)

4. Интеллектуальная собственность

Jorenza не претендует на владение контентом репозиториев, анализируемых через Сервис. Все права на код и файлы остаются у их авторов.

Бренд «Jorenza», логотип, дизайн интерфейса и уникальные алгоритмы генерации URL являются интеллектуальной собственностью команды Jorenza.

5. Ограничение ответственности

Сервис предоставляется «как есть» без каких-либо гарантий. Jorenza не несёт ответственности за:

  • Содержимое анализируемых репозиториев
  • Изменения в GitHub API, влияющие на работу Сервиса
  • Временную недоступность raw.githubusercontent.com
  • Потерю данных, хранящихся в localStorage браузера

6. Изменение условий

Jorenza оставляет за собой право изменять настоящие Условия. О существенных изменениях будет сообщено на главной странице Сервиса. Продолжение использования Сервиса после изменений означает принятие новых условий.
