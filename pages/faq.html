FAQ

Частые вопросы

Всё что ты хотел знать о Jorenza, но боялся спросить.

Общее

Что такое Jorenza?▾
Jorenza — платформа для мгновенного деплоя публичных GitHub репозиториев. Вставляешь ссылку на репо — получаешь красивый уникальный URL для своего проекта с прямыми ссылками на все файлы через raw.githubusercontent.com.
Это бесплатно?▾
Базовый план полностью бесплатный. Он включает анализ репозитория, генерацию уникального URL и короткой ссылки. Платные планы добавляют кастомные домены, аналитику и приоритетный доступ.
Нужна ли регистрация?▾
Нет! Jorenza работает без регистрации и без токенов GitHub. Просто вставляй ссылку и получай результат. История деплоев сохраняется локально в браузере.
Чем Jorenza отличается от Vercel или Netlify?▾
Vercel и Netlify — полноценные хостинг-платформы с CI/CD, серверными функциями и сборкой проекта. Jorenza — легковесная альтернатива для быстрого просмотра и шеринга проектов: каждый файл маппится напрямую на raw.githubusercontent.com, никакой сборки не происходит. Идеально для демо, портфолио и шеринга с друзьями.
Как выглядит URL проекта?▾
Каждый деплой получает уникальный URL вида jorenza.app/p/xK9mN2pQaB — 10 случайных символов из букв и цифр. Дополнительно генерируется короткая ссылка jrn.to/xK9mN2 для удобного шеринга.

Технические вопросы

Работает ли с приватными репозиториями?▾
Нет. Jorenza работает только с публичными репозиториями GitHub — без токенов авторизации. Это сделано намеренно: простота и безопасность важнее.
Что такое raw ссылки на файлы?▾
raw.githubusercontent.com — CDN GitHub для прямого доступа к содержимому файлов репозитория. Jorenza строит карту всех файлов вашего проекта и предоставляет прямые ссылки вида https://raw.githubusercontent.com/owner/repo/branch/path/to/file.js
Поддерживается ли React / Vue / Next.js?▾
Jorenza определяет тип проекта автоматически по структуре файлов. Для статических HTML проектов сайт откроется сразу. Для React/Vue/Next.js Jorenza предоставляет полную карту исходников с raw-ссылками — идеально для просмотра кода и шеринга.
Есть ли лимиты на количество файлов?▾
В дереве файлов отображается до 150 файлов. Для больших репозиториев это ограничение введено для производительности интерфейса. GitHub API также имеет лимит — 60 запросов в час без токена.
Как долго хранится деплой?▾
История деплоев сохраняется в localStorage вашего браузера. Деплои не удаляются автоматически — они хранятся пока вы не очистите данные браузера. В будущем планируется облачное хранение с аккаунтами.

Шеринг

Как поделиться проектом с друзьями?▾
После деплоя Jorenza генерирует короткую ссылку jrn.to/xxxxxx. Её можно отправить в Telegram, Twitter, VK или любой мессенджер. При переходе по ссылке откроется лендинг вашего проекта.
Что увидит человек перейдя по короткой ссылке?▾
Откроется лендинг проекта с названием репозитория, описанием, используемыми технологиями, статистикой (звёзды, форки) и прямыми ссылками на файлы. Если проект — статический HTML, будет возможность открыть сам сайт.
